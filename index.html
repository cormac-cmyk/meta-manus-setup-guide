Create Your Meta Developer App
This app is what connects Manus to your Meta Ads account. You'll select the Marketing API use case to get the right permissions automatically.

⚡ Quick Start — Short-Lived User Token
⏱️ This token expires in approximately 60 minutes. It's great for testing your connection and getting familiar with what Manus can do. For ongoing automation, use the Advanced path to generate a permanent token.
  • ✓
    Go to the Graph API Explorer at developers.facebook.com/tools/explorer/
  • ✓
    In the top-right dropdown, select your app from the "Meta App" list
  • ✓
    Click "Generate Access Token" — a Facebook login popup will appear
  • ✓
    Add the following permissions before generating:
    ads_management ads_read business_management pages_show_list pages_read_engagement
  • ✓
    Click "Generate Access Token" — log in and grant permissions when prompted
  • ✓
    Copy the generated token and save it somewhere accessible — you'll paste it into Manus next
    Remember: this token expires in ~60 minutes. Regenerate it each session if needed.

🔐 Advanced — System User Token (Permanent, Never Expires)
🔐 System User tokens do not expire based on time. They represent a server or software identity rather than a personal user — making them perfect for ongoing automation with Manus. They can only be invalidated if you change app permissions, delete the app, or modify business ownership.
Step A — Create a System User in Business Manager
  • ✓
    Go to business.facebook.com/settings/ and log in
  • ✓
    In the left sidebar, click Users → System Users
  • ✓
    Click "Add" — enter a name (e.g. Manus API User)
  • ✓
    Set the role to Admin — click "Create System User"
    Admin role is required for full ads management access including creating and pausing ads.
Step B — Assign Your App to the System User
  • ✓
    Select the System User you just created — click "Add Assets"
  • ✓
    Choose "Apps" from the asset type list — select the Meta App you created in Phase 2
  • ✓
    Enable "Manage App (Full Control)" — click "Save Changes"
Step C — Assign Your Ad Account to the System User
  • ✓
    Click "Add Assets" again — choose "Ad Accounts"
  • ✓
    Select your Ad Account — enable "Manage Ad Account" (full control)
  • ✓
    Click "Save Changes"
Step D — Generate the Permanent Token
  • ✓
    Select your System User — click "Generate New Token"
  • ✓
    Select your App from the dropdown
  • ✓
    Enable ALL of the following permissions:
    ads_management ads_read business_management pages_show_list pages_read_engagement read_insights
  • ✓
    Set token expiry to "Never" — click "Generate Token"
  • ✓
    Copy and save your token immediately — Meta will NOT show it again after you close this window
    Store it in a password manager (1Password, Bitwarden, etc.) — you'll need it for the Manus MCP configuration.
✅ You now have a permanent System User token. This token will keep Manus connected to your Meta Ads account indefinitely without needing to re-authenticate.

Troubleshooting & FAQs
Common issues and how to fix them.
▼
My token expired — how do I get a new one? ▼
If you used the Quick Start path, go back to the Graph API Explorer (developers.facebook.com/tools/explorer/) and generate a new token. If you used the Advanced System User path, your token should never expire — but if it has been invalidated, go to Business Manager → System Users and generate a new token for the same System User.
I'm getting "Permission Denied" errors in Manus ▼
Check that: (1) Your System User has the "Admin" role in Business Manager. (2) The System User has been assigned to the correct Ad Account with "Manage Ad Account" access. (3) Your token includes all the required permissions: ads_management, ads_read, business_management, pages_show_list, pages_read_engagement.
Manus can't see my ad account ▼
Make sure the ad account has been added as an asset to your System User in Business Manager. Go to Business Settings → System Users → select your user → Add Assets → Ad Accounts → select your account and enable full access.
Do I need App Review to use this? ▼
No. App Review is only required if people outside your business will log in and use your app. For connecting Manus to your own ad account (internal use), you do not need App Review. You can use the app in Development Mode indefinitely for your own accounts.
I've hit the app limit (15 apps) ▼
You can have a maximum of 15 apps not connected to a verified business portfolio. To resolve this: (1) Connect a verified business portfolio to existing apps, (2) Delete old or unused apps (note: archived apps still count), or (3) Remove yourself as admin/developer from apps you no longer need.
My System User token was invalidated unexpectedly ▼
System User tokens can be invalidated if: (1) App permissions were changed, (2) The Meta App was edited or removed, (3) Page or ad account access was revoked, or (4) Business ownership was modified. Generate a new token from Business Manager → System Users → Generate New Token.
